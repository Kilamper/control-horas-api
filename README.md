# Control de Horas

## Contributing

Please check out the [CONTRIBUTING.md](./CONTRIBUTING.md) document to learn about the contribution process.

## Getting Started

### Clone the repository

```bash
git clone <git url>
```

Enter the project directory

```bash
cd control-horas-api
```

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

## Project Structure
